# 🎨 Random Gradient Generator

A modern and interactive **Random Gradient Generator** built with **HTML, CSS and vanilla JavaScript**.

Generate beautiful CSS gradients instantly, preview them in real time and copy the code with a single click.

---

## 🚀 Live Demo

👉 https://johnyisbackk.github.io/js-random-gradient-generator/

---

## 🎮 Features

- Generate random gradient colors
- Real-time gradient preview
- Direction selector (angle & directions)
- Display of HEX color values
- Copy CSS gradient to clipboard
- Clean and modern dark UI
- Fully responsive design

---

## 🛠 Built With

- HTML5
- CSS3 (Modern UI, gradients, glassmorphism)
- Vanilla JavaScript

---

## 🎯 How It Works

1. Click **Generate Gradient**
2. Two random colors are created
3. Gradient is applied instantly to preview
4. Choose gradient direction from dropdown
5. Copy the CSS code with one click
